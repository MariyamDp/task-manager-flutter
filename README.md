# DB Task Manager

DB Task Manager is a Flutter sample app demonstrating SOLID principles and proper Dependency Injection using `get_it` and `injectable`.


## Quick Links

- Detailed developer docs: [docs/DOCUMENTATION.md](docs/DOCUMENTATION.md)
- Architecture diagram (PlantUML): [docs/ARCHITECTURE.puml](docs/ARCHITECTURE.puml)

## Getting Started (quick)

Clone, fetch packages and run:

```bash
git clone <repo-url>
cd DB_task_manager
flutter pub get
flutter run
```
